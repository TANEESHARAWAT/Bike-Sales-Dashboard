# Bike-Sales-Dashboard
Excel dashboard project analyzing bike sales data using Pivot Tables and Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning

## Dashboard Features
- Customer age analysis
- Income analysis
- Purchase comparison
- Customer commute analysis
- Interactive filters
